## Learning Objectives
At the end of this project, you are expected to be able to [explain to anyone](https://fs.blog/feynman-learning-technique/), without the help of Google:

### General
- Understand the Dart syntax and structure.
- Learn how to declare variables and use basic data types (int, double, String).
- Use control flow statements like if, else.
- Practice console input/output using print().
- Understand the role of the assert statement to validate conditions.
- Get familiar with the main() function as the entry point of a Dart program
